# Doctrine

## Purpose

Use this system to reduce large mistakes, execution drift, and memory distortion during volatility. The goal is not forecasting accuracy. The goal is disciplined decisions that can be inspected later.

## Non-Negotiables

- Put new ideas into the watchlist before formal trade review.
- State the thesis in plain language.
- State the invalidation condition before execution.
- Match position size to confidence and downside.
- Distinguish a process error from a bad short-term outcome.
- Turn repeated mistakes into named behavior patterns.

## Interpretation Rules

- Treat missing thesis, invalidation, or sizing logic as a serious weakness.
- Treat emotional urgency without new information as a warning sign.
- Treat a change in position size as a decision that requires its own review.
- Prefer delay or size reduction over forced conviction.

## Review Philosophy

- Judge the process first.
- Judge the result second.
- Record a lesson only if it can be turned into a future rule or checklist item.
